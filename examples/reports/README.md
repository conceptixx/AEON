# AEON Example Reports

This directory contains example reports generated by the AEON installation process.

## Files

- **example-report.md** - Markdown report example (for documentation)
- **example-report.json** - JSON report example (for automation)

## What These Show

These examples demonstrate what users will see after a successful AEON cluster installation. The actual reports are generated in `/opt/aeon/reports/` on the installation device.

## Formats

AEON generates reports in multiple formats:
- **Terminal** - Colored, formatted display during installation
- **Markdown** - GitHub-ready documentation
- **JSON** - Machine-readable for automation
